# Glad Animal Movements — Website

One-page portfolio site for [Glad Animal Movements](https://www.linkedin.com/in/conor-donelan-a58854247/), the videography studio of Conor Donelan (Dublin & Wicklow, Ireland).

Plain HTML/CSS/JS — no build step. Open `index.html` in a browser, or serve the folder with any static file server.

## Structure

- `index.html` — the whole site (hero, about, services, work, clients, contact)
- `css/style.css` — styling (nature palette: fern green, sunlit gold, clay)
- `js/main.js` — mobile nav, scroll-reveal animations
- `assets/` — hero drone video, favicon, client logos

## Editing

All text lives in `index.html`. To swap the hero video, replace `assets/hero-lighthouse.mp4` (H.264 MP4, 1080p, no audio needed — it plays muted).
